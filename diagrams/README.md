# Biblioteca de diagramas

Guardá acá los archivos `.txt` de diagramas (el DSL de "diagrama").

- Cada archivo `*.txt` de esta carpeta aparece automaticamente en el menú
  **importar (prompt → diagrama)** de la app (Ctrl+I), listo para abrir con un click.
- Si agregás un archivo nuevo mientras la app está abierta, **recargá** la pagina
  para que la biblioteca lo tome.
- El nombre que se muestra es el `title:` del diagrama (si tiene) o el nombre del archivo.

Las sesiones de Claude que generen diagramas deben dejar el `.txt` en esta carpeta
(ver `public/prompt-to-diagram.txt`).
